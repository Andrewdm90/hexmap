Grid Map Editor Help

Version 18 - 13/04/2025

Instructions on how to use the editor:

  • Use the Grid Settings to change map size, shape, and layout. Click the header to collapse/expand.
  • Select tools from the tabs below the grid settings.
  • Paint Tab: Click cells to color them or erase color. Use color picker, hex input, or eyedropper.
  • Icon Tab: Upload SVG/PNG/JPG icons. Select one from the gallery. Click map to stamp. Use sliders to adjust size and rotation *before* stamping. Click 'Delete Icon' then click icons on the map to remove them. Click the red '‚ùå' on gallery icons to remove them from the gallery.
  • Text Tab: Enter text, set size/color using picker, hex, or eyedropper. Click map to stamp. Click 'Delete Text' then click text on map to remove.
  • Draw Tab: Select color/thickness using picker, hex, or eyedropper. Click and drag on the map to draw lines. Use eraser to remove parts of drawings.
  • Options Tab: Toggle cell UI visibility, size, and vertical position. Save/Load map state (.json). Export notes (.html) or the map view (.png).
  • No Tool Button: You MUST click 'No Tool' to deselect other tools before you can click on a cell's number to add or edit notes. The currently selected tool is shown next to this button.
  • Notes: When 'No Tool' is active, click a cell number (e.g., 0101) to open the notes editor for that cell. A book icon (üìì) indicates existing notes. Use the toolbar above the notes area for formatting (Bold, Italic, Underline, Bullets, Alignment, Insert Image from URL).
  • Eyedropper (üíß): Click the eyedropper button next to a color input, then click anywhere on the page to pick a color (Not available in all browsers).
  • Dark Mode: Use the toggle switch in the top right.
